# HelloBot

Простой Telegram-бот на Node.js. На любое сообщение отвечает: «Привет, я бот!»

## Запуск

1. В файле `.env` задайте токен: `BOT_TOKEN=ваш_токен` (без пробелов вокруг `=`).
2. Установите зависимости (если ещё не установлены):
   ```powershell
   npm install
   ```
3. Запустите бота:
   ```powershell
   npm start
   ```
